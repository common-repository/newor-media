== Description ==

Newor Media plugin simplifies the process of adding Newor Media ads to the WordPress blog.

#### Features and Advantages
 
* Dynamic ads.txt file
 * Seamlessly automate the updating of ads.txt files, eliminating manual work and saving time
* Automated header script
 * Allows for quick addition of header script to the site, no more manual placement
* Automated sticky footer ad
 * Allows for quick insertion of sticky footer ads onsite
* Automated interstitial ad
 * Allows for speedy insertion of interstitial ads onsite
* Incontent ads
 * Easily add incontent, higher impact, and higher earnings units with speed and ease of use
 
== Installation ==
 
#### From within WordPress

1. Visit 'Plugins > Add New'
2. Search for 'Newor Media'
3. Activate 'Newor Media' for WordPress from your 'Plugins' menu from WordPress.
4. Visit the new 'Newor Media' menu to configure the ads.

#### Manually

1. Upload the newormedia/ folder to the /wpcontent/plugins/ directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress. 
3. Visit the new 'Newor Media' menu to configure the ads.
